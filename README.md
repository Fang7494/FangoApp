# Fango A++

A clean, modern Meteor-inspired Minecraft cheat client for Fabric 0.19.1 / MC 1.21.1.

## Features
- 35+ modules across 6 categories: Combat, Player, Movement, Render, World, Misc
- Meteor Dark theme + Classic Dark theme
- Two-panel ClickGUI (Right Shift to open)
- Full settings system: sliders, toggles, dropdowns, checkboxes
- HUD with ArrayList, coordinates, FPS, ping
- Notifications system
- AntiBot, Blink, Freecam, Scaffold, ESP, Tracers, Xray, and more

## Build
```
./gradlew build
```

## Requirements
- Java 21+
- Minecraft 1.21.1
- Fabric Loader 0.19.1
- Fabric API

## Key Bind
- **Right Shift** — Open/Close GUI
